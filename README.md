# Card Generator
Static theme generator for business cards, based on inputs to config file. This source have 4 themes.

### Configuration 

cards.json

[
  {
    "style":"card_1"
  },
  {
    "style":"card_2"
  },
  {
    "style":"card_3"
  },
  {
    "style":"card_4"
  }
]

Add card folder(which contains a HTML file & CSS file) name as a key in config, to add it as a new style.
